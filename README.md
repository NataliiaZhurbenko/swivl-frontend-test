## Running locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
git clone swivl-frontend-test
cd swivl-frontend-test
npm install
$(npm bin)/static
```

The app should now be running on [127.0.0.1:8080/dist](http://127.0.0.1:8080/dist).
